Abstract

In today’s fast-moving academic and professional environments, productivity tools have become essential for managing daily tasks. However, most existing task managers focus only on efficiency and scheduling, overlooking an important human factor—mood. TodoMood is a mood-based task management system that addresses this gap by integrating emotional awareness into task planning. The system is designed to adapt tasks according to the user’s current emotional state, helping them stay productive without feeling burdened.
Developed using HTML, CSS, and JavaScript, and supported by simple Natural Language Processing (NLP) techniques, TodoMood enables users to record their mood and receive task suggestions that match it. For example, when a user selects “Tired,” the system highlights smaller, less demanding tasks, whereas choosing “Focused” prioritizes high-importance or cognitively challenging work. This adaptive approach reduces decision fatigue and supports mindful task selection, particularly for students and professionals who manage fluctuating workloads.
The interface emphasizes simplicity and usability, offering features such as animated task cards, mood-based color themes, and completion tracking. Planned backend integration with Firebase will allow real-time data storage, synchronization, and retrieval across devices. Although the current NLP model is rule-based with keyword tagging, future improvements aim to include voice input, mood analytics, and advanced contextual recommendations.
By combining productivity with emotional intelligence, TodoMood demonstrates how simple technologies can be used creatively to design human-centered solutions. The project highlights how recognizing the emotional dimension of work can improve engagement, reduce cognitive overload, and promote a balanced approach to personal productivity.
